🌌 Cosmic Catch

A fast-paced cosmic catching game with social features, levels, power-ups, and global leaderboards.

🎮 Features

  • 3-screen flow: Sign In → Menu → Gameplay
  • Social sign-in (Facebook, Google, Guest)
  • 10 progressive levels with increasing difficulty
  • Power-ups: Shield 🛡️ protects from bombs
  • Combo system for bonus points
  • Friend challenges & global leaderboards
  • Background music & sound effects
  • Mobile-optimized touch controls

🚀 Getting Started

# Clone the repository
git clone https://github.com/yourusername/cosmic-catch.git

# Open in browser
open index.html
